Road Accident Analysis

  • Tech Stack: Python, Tableau, MySQL, Excel and Git
  • Tableau Public URL: Project Link

This project involved an in-depth analysis of road accident data to uncover key trends and identify contributing risk factors. The aim was to support data-driven decision-making for improving road safety and resource optimization.

Key Highlights:

Data Analysis: Performed a comprehensive analysis of accident data, including time, location, weather conditions, and other contributing factors. Identified patterns and high-risk areas using statistical techniques and exploratory data analysis (EDA).

Dashboard Development: Designed interactive dashboards in Tableau to visualize critical metrics such as: Accident frequency by time of day, week, and season. Geographic hotspots for accidents using heatmaps. Analysis of accident causes, vehicle types, and demographics of those involved. Enabled users to filter and drill down into specific categories, such as location or accident type. Insights and Recommendations:

Delivered actionable insights to stakeholders: Highlighted peak accident times and high-risk zones. Suggested targeted interventions, such as better signage, improved lighting, or enhanced traffic enforcement in critical areas. Provided recommendations for resource allocation to mitigate accident rates and improve response times.

Impact: Facilitated improved road safety measures through data-driven policies. Optimized resource deployment to reduce accidents and enhance emergency response efficiency.
